# Growth Graph Generator

This development generates generic graphs and serves as an example of practical solutions.

A Growth Graph is a graph that evolves step-by-step through deterministic or rule-based expansion.<br>
Each transformation step takes an initialGraph as input, applies a set of growth rules, and produces a new graph state.<br> 
The output of one step becomes the initialGraph of the next step, forming a growth pipeline.<br>
In this solution, the input graphs remain unchanged. Only additional edges are added.<br>
